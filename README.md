# Comfyui-MK-Theme

一个面向 ComfyUI 的轻量前端插件，包含画布对齐吸附、分组美化，以及一个可拖动的 MK 悬浮工具栏。

## 功能

- 顶部工具栏提供对齐吸附开关。
- 拖动节点或分组时，可吸附到附近节点和分组的边缘、中心线。
- 支持多选节点作为整体一起吸附移动。
- 拖动时按住 `Shift` 可临时跳过吸附。
- 可在设置中调整吸附距离。
- 分组显示更清晰的标题栏、边框、高亮和选中效果。
- 打开 ComfyUI 后显示可拖动的 `MK` 悬浮图标。
- 悬浮工具栏支持复制/粘贴节点大小。
- 悬浮工具栏支持复制/粘贴节点颜色。
- 悬浮工具栏支持用完整色域色盘分别设置节点标题栏颜色和节点主体颜色。

## 使用方式

把本仓库放入 ComfyUI 的 `custom_nodes` 目录，然后重启 ComfyUI。

```text
ComfyUI/custom_nodes/Comfyui-MK-Theme
```

启动后，ComfyUI 会自动加载 `js` 目录下的前端扩展。

## 悬浮工具栏

选中一个节点后，把鼠标移动到画布上的 `MK` 悬浮图标，右侧会展开工具栏。

- `复制大小`：复制当前选中节点的宽高。
- `粘贴大小`：把复制的宽高应用到当前选中节点。
- `复制颜色`：复制当前选中节点的标题栏颜色和主体颜色。
- `粘贴颜色`：把复制的颜色应用到当前选中节点。
- `菜单栏` 色盘：设置当前选中节点的标题栏颜色。
- `节点` 色盘：设置当前选中节点的主体背景颜色。

粘贴和色盘设置支持多选节点。

## 文件结构

- `__init__.py`：ComfyUI 插件入口，暴露 `js` 前端扩展目录。
- `js/align/index.js`：MK-Theme 对齐参考线和吸附功能。
- `js/floating_icon.js`：MK 悬浮工具栏功能。
- `js/linear_theme.js`：独立集成的 Group-Styler 分组美化功能。
- `js/assets/icons/ui/align-center-v.svg`：对齐工具栏图标。
- `config.json`：Group-Styler 保留配置资产。
- `linear_dark.json`：Group-Styler 保留主题资产。

## 说明

本插件保留了 MK-Theme 对齐吸附功能和 Group-Styler 分组美化功能的独立边界，同时新增独立的悬浮工具栏模块，方便在画布中快速处理节点尺寸和颜色。
